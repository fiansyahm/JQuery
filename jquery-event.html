Jquery Event

1. Menghilangkan judul dengan satu klik

Halo Semua

$(this).hide() - Menyembunyikan elemet sekarang.
$("h1").hide() - menyembunyikan element h1.
$(".test").hide() - menghilangkan element class=".test" ".
$("#test").hide() - menghilangkan element dengan id="test".
